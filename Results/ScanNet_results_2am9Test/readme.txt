Description of the outputs:
annotated_[PDBID].pdb/cif: Structure file with the binding site probability written in the B-factor field.
predictions_[PDBID].csv: CSV file containing the binding site probability, together with amino acid indices and type.
annotated_[PDBID].cxc: Short ChimeraX script for opening and coloring the structure based on interface probability. Usage: right-click -> Open with ChimeraX.
annotated_[PDBID].py: Short Chimera script for opening and coloring the structure based on interface probability. Usage: right-click -> Open with Chimera.
